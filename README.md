## Project Overview
This project is a 12V AC to DC converter designed using KiCad as my first PCB design trial. The circuit uses four 1N4007 diodes arranged in a full-wave bridge rectifier configuration to convert the AC input into pulsating DC. A 1000µF filter capacitor smooths the rectified output by reducing ripple voltage and providing a more stable DC level. An LED indicator with a current-limiting resistor is included to show the presence of output voltage. This project helped me understand the fundamentals of rectification, filtering, and basic PCB layout design using KiCad.
